Чтобы собрать проект в вашей системе, вам нужно установить необходимые для него зависимости. 
Вы также можете установить большинство зависимостей Qt через их установщик.
Если вы раньше разрабатывали приложения Qt, возможно, они у вас уже есть.

Зависимости
Compile-time
 -Qt >= 5.9
   -Development tools
 -GCC >= 7.4
 -CMake >= 3.13
Run-time
 -Qt
 -SVG
Optional
 -Git
 -OpenSSL
 -CA Certificates

В этом проекте используется система сборки CMake, поэтому вам необходимо установить ее для сборки 
проекта (в большинстве дистрибутивов Linux она доступна в стандартных репозиториях в виде пакета 
под названием cmake). Если ваш дистрибутив содержит слишком старую версию CMake 
(например, Ubuntu 18.04), вы можете скачать ее на официальном сайте.

Также вы можете открыть и собрать/отладить проект в C++ IDE. Например, в Qt Creator вы можете просто 
открыть CMakeLists.txt через «Открыть файл» или «Проект» в меню после установки CMake в вашей системе.
После установки всех зависимостей выполните следующие команды в корневом каталоге исходного кода:
 mkdir build
 cd build
 cmake ../
 make
Когда команда make завершена, вы можете запустить flameshot из папки project_folder/build/src.
